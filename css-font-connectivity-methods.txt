1 :- External CSS
  Create one file with .css extension e.g style.css
2 :- Connecting with cdnjs.com
  -Go to the cdnjs.com
  -get the url which we want
  -combine it with the link tag
  e.g :- <link href="https://cdnjs.cloudflare.com/ajax/libs/normalize/5.0.0/normalize.css" rel="stylesheet">
3 :- Property of the css
    -background-color
    -background
    -border
    -height
    -width
    -position
    -top
    -bottom
    -left
    -margin
    -padding
    -float
    -align-self
4 :- Creating class in css
  - Synatx
  .class_name{
    property : value;
  }
  -e.g
  .sidebar{
    height:100%;
    width:30%;
  }
